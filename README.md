# MaterialManager-物料管理软件

#### 介绍
简单的物料管理软件，对物料进行管理，`本人主要用于管理设备配件`

#### 功能介绍
1. 物料出库，入库。
2. 物料挂账人转变。
3. 可按时间、物料，查询 出库、入库、转账记录。
4. 每个人管理自己名下的物料。
